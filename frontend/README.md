# 小月 🌙 - 情感陪伴玩偶前端

基于 React 18 + Vite 构建的情感陪伴玩偶前端界面。

## 快速开始

```bash
npm install
npm run dev
```

访问 `http://localhost:5173`

## 环境配置

确保 `.env.local` 文件存在：
```bash
VITE_API_URL=http://localhost:8080
```

## 核心功能

- ✅ 玩偶表情切换 (5种表情)
- ✅ 点击玩偶弹跳动画
- ✅ 用户消息发送和 AI 回复
- ✅ 自动滚动到最新消息
- ✅ 剩余消息数提示
- ✅ 响应式布局

详见 `FRONTEND-DEV-GUIDE.md`
